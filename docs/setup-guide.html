⚙️ セットアップガイド

Nord Final 採用の 17 要素のうち、URL 貼るだけ以外で「事前セットアップが必要な 4 つ」の手順をまとめます。

1. ⏱ Wakatime

所要時間: 10 分 + コーディング 1-2 週間 (データ蓄積)

手順

  1. https://wakatime.com/signup で GitHub サインアップ
  2. Settings → API Key をコピー
  3. 使用エディタの拡張機能をインストール:
    • Cursor / VS Code: Marketplace で WakaTime 検索 → install → Cmd+Shift+P → WakaTime: API Key で貼付
    • Vim / Neovim: :Plug 'wakatime/vim-wakatime'
    • JetBrains: Plugins → WakaTime install
    • Ghostty / iTerm: brew install wakatime-cli + terminal 設定
  4. README に貼り付ける URL は API Key 不要 (公開 stats 経由):
    https://github-readme-stats.vercel.app/api/wakatime?username=cardene777&theme=nord&hide_border=true&layout=compact
  5. Wakatime Settings で「Display code time publicly」を ON にする

注意点

  • 初日は "No data" になる。1-2 週間使ってから貼ると見栄えが良い
  • 言語別・OS 別・Editor 別など複数 layout がある

2. 🌳 3D Profile Contrib

所要時間: 5 分 (workflow 追加のみ)

手順

  1. .github/workflows/profile-3d.yml を作成:
name: GitHub-Profile-3D-Contrib
on:
  schedule:
    - cron: "0 0 * * *"  # 毎日 0:00 UTC
  workflow_dispatch:
  push:
    branches: ["main"]

jobs:
  build:
    runs-on: ubuntu-latest
    permissions:
      contents: write
    name: generate-github-profile-3d-contrib
    steps:
      - uses: actions/checkout@v4
      - uses: yoshi389111/github-profile-3d-contrib@0.7.1
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
          USERNAME: ${{ github.repository_owner }}
          SETTING_JSON: profile-3d-contrib/settings.json
      - name: Commit & Push
        run: |
          git config user.name github-actions[bot]
          git config user.email 41898282+github-actions[bot]@users.noreply.github.com
          git add -A .
          git diff --quiet HEAD || git commit -m ":zap: Update 3D contrib"
          git push
  1. profile-3d-contrib/settings.json を作成 (Nord 風配色):
{
  "type": "season",
  "backgroundImageUrl": "",
  "fileName": "profile-night-rainbow.svg",
  "settings": [{ "type": "night-rainbow", "backgroundColor": "#2e3440" }]
}
  1. Settings → Actions → General → Workflow permissions → "Read and write permissions" を ON
  2. 初回は Actions タブから手動実行 (workflow_dispatch)
  3. README に貼る:
    ![3D Contrib](./profile-3d-contrib/profile-night-rainbow.svg)

3. 📐 Metrics Pin

所要時間: 10 分 (PAT 発行 + workflow 追加)

手順

  1. Personal Access Token (Classic) を発行:
    • Settings → Developer settings → Personal access tokens → Tokens (classic) → Generate new token (classic)
    • Scope: repo, read:user, read:org を付与
    • Token をコピー
  2. cardene777/cardene777 リポの Settings → Secrets and variables → Actions → New repository secret:
    • Name: METRICS_TOKEN
    • Value: 発行した PAT
  3. .github/workflows/metrics.yml を作成:
name: Metrics
on:
  schedule:
    - cron: "0 0 * * *"
  workflow_dispatch:
  push:
    branches: ["main"]

jobs:
  metrics:
    runs-on: ubuntu-latest
    permissions:
      contents: write
    steps:
      - uses: lowlighter/metrics@latest
        with:
          filename: metrics.svg
          token: ${{ secrets.METRICS_TOKEN }}
          base: header, activity, community, repositories, metadata
          plugin_habits: yes
          plugin_habits_facts: yes
          plugin_habits_charts: yes
          plugin_topics: yes
          plugin_topics_mode: icons
          plugin_followup: yes
          plugin_achievements: yes
          plugin_achievements_threshold: B
          config_timezone: Asia/Tokyo
          config_padding: 4 + 1%
          config_octicon: yes
          config_order: base.header, base.activity, plugin_habits, plugin_topics, plugin_followup, plugin_achievements
  1. Actions タブから手動実行 → metrics.svg がリポルートに生成される
  2. README に貼る:
    ![Metrics](./metrics.svg)

4. 🐍 Snake Nord 配色化

所要時間: 2 分 (snake.yml の color 値を 1 行修正)

手順

  1. .github/workflows/snake.yml を編集:
# Before
- uses: Platane/snk@v3
  with:
    github_user_name: ${{ github.repository_owner }}
    outputs: |
      dist/snek.svg
      dist/snek-dark.svg?palette=github-dark
      dist/snek.gif?color_snake=orange&color_dots=#bfd700,#8ebd00,#5a8700,#2e4d00,#0f1f00

# After (Nord 配色)
- uses: Platane/snk@v3
  with:
    github_user_name: ${{ github.repository_owner }}
    outputs: |
      dist/snek.svg?palette=github-light
      dist/snek-dark.svg?color_snake=#88c0d0&color_dots=#3b4252,#a3be8c,#88c0d0,#81a1c1,#eceff4
      dist/snek.gif?color_snake=#88c0d0&color_dots=#3b4252,#a3be8c,#88c0d0,#81a1c1,#eceff4
  1. commit + push → 翌日 0:00 に自動再生成 (もしくは Actions タブから手動実行)
